📘 Chapter 1: Second Degree Equation

Lesson Summary

A quadratic equation in one variable x is of the form: ax² + bx + c = 0, with a ≠ 0.

Incomplete quadratic equations are special cases where one term is missing:

  • ax² + bx = 0
  • ax² + c = 0
  • ax² = 0

To solve a quadratic equation, we use the discriminant Δ = b² − 4ac:

  • Δ > 0 → two distinct real solutions.
  • Δ = 0 → one real double solution.
  • Δ < 0 → no real solution (complex solutions).

Part 3: Equations Leading to Quadratic Form

10 Solved Examples

1) (x + 1)(x − 2) = 3 → x² − x − 5 = 0 → Δ = 21 → x = (1 ± √21)/2

2) (x − 1)² = 4x → x² − 6x + 1 = 0 → Δ = 32 → x = 3 ± 2√2

3) (2x + 3)(x − 1) = 5 → 2x² + x − 8 = 0 → Δ = 65 → x = (−1 ± √65)/4

4) (x + 2)² = x + 10 → x² + 3x − 6 = 0 → Δ = 33 → x = (−3 ± √33)/2

5) (x − 4)(x + 1) = 2 → x² − 3x − 6 = 0 → Δ = 33 → x = (3 ± √33)/2

6) (x + 5)(x − 2) = x → x² + 2x − 10 = 0 → Δ = 44 → x = (−2 ± √44)/2

7) (x − 3)² = 2x → x² − 8x + 9 = 0 → Δ = 28 → x = (8 ± √28)/2

8) (x + 4)(x − 1) = 7 → x² + 3x − 11 = 0 → Δ = 53 → x = (−3 ± √53)/2

9) (x − 2)(x − 5) = 6 → x² − 7x + 4 = 0 → Δ = 33 → x = (7 ± √33)/2

10) (x + 1)(x + 2) = x² + 5 → x² + 3x + 2 = x² + 5 → 3x + 2 = 5 → x = 1
